# Figura PWA - Installation Guide

## What's New: Progressive Web App (PWA)

Figura is now a Progressive Web App! You can install it on your mobile device and use it just like a native app.

## Features

✅ **Installable** - Add to home screen on iOS and Android
✅ **Offline Ready** - Service worker caches assets for offline use
✅ **Camera Access** - Take photos directly from the app
✅ **Fast Loading** - Optimized performance with caching
✅ **App-like Experience** - Full-screen standalone mode

## How to Install on Mobile

### iOS (Safari)
1. Open https://your-app-url.com in Safari
2. Tap the Share button (square with arrow)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add" in the top right corner
5. The Figura app icon will appear on your home screen

### Android (Chrome)
1. Open https://your-app-url.com in Chrome
2. Tap the menu (three dots) in the top right
3. Tap "Add to Home screen" or "Install app"
4. Confirm by tapping "Install"
5. The Figura app icon will appear on your home screen

## Development

To run the PWA locally:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
npm run preview
```

## PWA Configuration

- **Manifest**: `/public/manifest.json` - App metadata and icons
- **Service Worker**: Automatically generated by Vite PWA plugin
- **Icons**: SVG-based icons in `/public/` directory
- **Cache Strategy**: Network-first for API calls, cache-first for static assets

## Camera Permissions

The app requires camera permissions to take photos. Make sure to:
- Grant camera access when prompted
- Use HTTPS in production (required for camera API)
- On iOS, Safari is required for full PWA support

## Technical Details

- **PWA Plugin**: vite-plugin-pwa
- **Workbox**: For service worker and caching strategies
- **Theme Color**: #a8c878 (sage green)
- **Background Color**: #faf6ee (warm linen)
- **Display Mode**: standalone (full-screen)
- **Orientation**: portrait (optimized for mobile)

## Notes

- For production deployment, ensure HTTPS is enabled
- Test installation on actual devices for best experience
- PNG icons (192x192, 512x512) recommended for better compatibility
  - Current setup uses SVG icons as fallback
  - To generate PNG icons, use: https://realfavicongenerator.net/

## Deployment

When deploying to production:

1. Build the app: `npm run build`
2. Deploy the `dist/` folder to your hosting service
3. Ensure HTTPS is enabled
4. Test PWA installation on mobile devices
5. Verify service worker is registered in DevTools

## Browser Support

- ✅ iOS Safari 11.3+
- ✅ Android Chrome 80+
- ✅ Samsung Internet 13+
- ⚠️ Desktop browsers (installable but not optimized for mobile experience)
